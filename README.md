To run this project locally in VS Code and see a live preview:

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### 2. Installation
Open your terminal in VS Code (`Ctrl + ` `) and run:
```bash
npm install
```

### 3. Run Development Server
Start the Vite dev server by running:
```bash
npm run dev
```

### 4. View Preview
- **In Browser:** Open [http://localhost:5173](http://localhost:5173).
- **In VS Code:** Install the **Live Preview** extension by Microsoft. Once installed, press `Ctrl + Shift + P`, type "Live Preview: Show Preview", and enter `http://localhost:5173`.
